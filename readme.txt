Boomerang is a feature request management system for WordPress.

== Description ==

Boomerang provides a simple interface for your users or customers to provide feedback about your goods or services. Users must be logged in to submit 'Boomerangs'.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin settings can be accessed via the 'Settings' menu in the administration area (either your site administration for single-site installs).
